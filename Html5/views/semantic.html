This paragraph is a child of main. The main content should not include any content that is repeated across pages (e.g. menus/site navigation).

Section content

Sections should always have a heading and should be used for a thematic grouping of content.

Stressing parts of text

b
Highlighting keywords.
strong
Indicating importance. e.g. Turn right at the next junction.
em
For words that would be pronounced with emphasis if read aloud. e.g. I am very unhappy with your behaviour!
i
For an alternate voice or mood. e.g. stage directions. Difficult to come up with a compelling use but see HTML5 Doctor for more information.

Definitions and descriptions

Description lists

A dl tag is used to create descriptions for terms. The relationship between terms and descriptions is many to many so a term can have more than one description and descriptions can relate to more than one term.

See Stressing parts of text and Code / Samp / Pre / Var for example uses.
