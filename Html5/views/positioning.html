Positioning

There are four different position values that can be applied to any element: static, relative, absolute, and fixed.

The default positioning is static, but if any other position value is used then you can adjust the position of the element using one or more of the following properties: top, right, bottom and left.

position: static (default)

Elements with static position are placed in the document based on the flow of the rest of the content.

position: relative;

With relative positioning, the element's position is adjusted relative to its static position. This div uses top: 50px; left: 30px; Notice that the content wraps relative to the static position, rather than the relative position.

position: absolute;

With absolute positioning, the element's position is not affected by the flow of the document. Instead it will always have the same position relative to the document and it will move when the user scrolls. This div uses top: 630px; left: 100px;

position: fixed;

With fixed positioning, the element's position is not affected by the flow of the document. Instead it will always have the same position relative to the window and it will not move when the user scrolls. This div uses top: 465px; left: 200px;
