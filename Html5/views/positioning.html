Positioning

Back to layout

There are four different position values that can be applied to any element: static, relative, absolute, and fixed.

The default positioning is static, but if any other position value is used then you can adjust the position of the element using one or more of the following properties: top, right, bottom and left.

position: static (default)

Elements with static position are placed in the document based on the flow of the rest of the content.

position: relative;

top: 50px; left: 30px;
With relative positioning, the element's position is adjusted relative to its static position. Notice that the content wraps relative to the static position, rather than the relative position.

position: absolute;

top: 900px; left: 100px;
With absolute positioning, the element's position is not affected by the flow of the document. Instead it will always have the same position relative to the document and it will move when the user scrolls.

absolute within absolute

top: 120px; left: -10px;
This absolute content is nested within another element with absolute position, so it is positioned relative to the parent.

position: fixed;

top: 465px; left: 200px;
With fixed positioning, the element's position is not affected by the flow of the document. Instead it will always have the same position relative to the window and it will not move when the user scrolls.

absolute within fixed

top: 200px; left: 200px;
This absolute content is nested within an element with fixed position, so it is positioned relative to the parent.

static within fixed

This static content is nested within an element with fixed position.

relative within fixed

top: 50px; left: 30px;
This relative content is nested within an element with fixed position.
